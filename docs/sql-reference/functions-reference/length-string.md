# LENGTH

Calculates the string length.

## Syntax
{: .no_toc}

```sql
LENGTH(<string>)
```

| Parameter  | Description                                |
| :---------- | :------------------------------------------ |
| `<string>` | The string for which to return the length. |

## Example
{: .no_toc}

```sql
SELECT LENGTH('abcd')
```

**Returns**: `4`
