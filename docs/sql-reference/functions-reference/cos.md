# COS

Calculates the cosine.

## Syntax
{: .no_toc}

```sql
COS(<exp>)
```

| Parameter | Description                                           |
| :--------- | :----------------------------------------------------- |
| `<exp>`   | Any expression that evaluates to a numeric data type. |

## Example
{: .no_toc}

```sql
SELECT
    COS(180);
```

**Returns**: `-0.5984600690578581`
