# TAN

Calculates the tangent.

## Syntax
{: .no_toc}

```sql
TAN(<val>)
```

| Parameter | Description                                                                                                          |
| :--------- | :-------------------------------------------------------------------------------------------------------------------- |
| `<val>`   | Valid values include column names, functions that return a column with numeric values, and constant numeric values.  |

## Example
{: .no_toc}

```sql
SELECT
    TAN(90);
```

**Returns**: `-1.995200412208242`
