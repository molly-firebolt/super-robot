# DROP ENGINE
Deletes an engine.

## Syntax

```sql
DROP ENGINE [IF EXISTS] <engine_name>
```

| Parameter       | Description                           |
| :--------------- | :------------------------------------- |
| `<engine_name>` | The name of the engine to be deleted. |
