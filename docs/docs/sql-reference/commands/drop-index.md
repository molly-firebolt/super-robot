# DROP INDEX
Deletes an index.

## Syntax

```sql
DROP [AGGREGATING | JOIN] INDEX [IF EXISTS] <index_name>
```

| Parameter      | Description                          |
| :-------------- | :------------------------------------ |
| `<index_name>` | The name of the index to be deleted. |
